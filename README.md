# Vijportfolio Next.js

This repository has been converted from a static HTML portfolio to a Next.js app.

## Run locally

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start development server:
   ```bash
   npm run dev
   ```
3. Open `http://localhost:3000`

## Notes

- Static assets are now served from `public/assets/` and `public/casestudies/`.
- The main site page is at `pages/index.js`.
- Global styles are in `styles/globals.css`.
